Theses Checker is an online tool that checks your thesis mainly for typographical errors. This tool searches for mistakes such as text overflow, wrong usage of hyphens, no section description and others. To find some mistakes, Theses Checker searches for keywords. Therefore the thesis should be either in Czech, English or Slovak; otherwise, some checks might not work as expected.
